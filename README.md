# Timely Chat

AI Chat 모듈을 위한 CDN 배포 저장소입니다.

## 최신 버전
- Production: **1.0.15**
- Staging: **1.0.15**

## 사용법

### Production
```html
<script src="https://cdn.jsdelivr.net/gh/timely-hub/timely-chat@p.1.0.15/index.js"></script>
```

### Staging
```html
<script src="https://cdn.jsdelivr.net/gh/timely-hub/timely-chat@stg.1.0.15/index.js"></script>
```

### TypeScript 지원

